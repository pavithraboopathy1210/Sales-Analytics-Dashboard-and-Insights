# Sales Data Analysis using Excel

## 📊 Project Overview
This project analyzes sales data to identify trends, patterns, and business insights.

## 🛠 Tools Used
- Microsoft Excel

## 🔍 Key Tasks
- Data Cleaning
- Data Visualization (Charts, Pivot Tables)
- Business Insights Generation

## 📈 Insights
- Identified top-performing products
- Analyzed region-wise sales
- Found seasonal trends

## 📌 Conclusion
This analysis helps in improving business decision-making and increasing sales performance.
